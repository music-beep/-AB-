> **项目概述：** 本项目基于 **290,000+** 条用户行为日志，从中选取2000条数据，针对“新版落地页能否提升 5% 点击转化率”的业务诉求，对电商网站新旧落地页进行 A/B 测试评估。执行双尾 Z 检验，计算得出 P-value = 0.4834 ( > 0.05 )，结合对照组 95% 置信区间 [9.06%, 
12.94%]，科学判定新版页面无显著增益。

---

## 📊 核心成果 (Executive Summary)

经过严谨的假设检验与数据验证，实验取得了显著的正向收益：

对照组 (Control) 转化率: 11.0000%
实验组 (Treatment) 转化率: 12.0000%
转化率相对提升: 9.0909%

========== A/B 测试检验结果 ==========
Z-statistic (Z统计量): 0.7009
P-value (P值): 0.4834
对照组 95% 置信区间: [9.0607%, 12.9393%]
实验组 95% 置信区间: [9.9859%, 14.0141%]

========== 最终结论 ==========
结论: P-value >= 0.05，结果不具有统计学显著性，无法证明新页面优于旧页面。

* **统计结论：** 0.4834 > 0.05$，接受原假设。新页面在 95% 置信水平与旧业面无显著增益。

## 🛠️ 分析方法与技术细节 

作为统计学背景的分析项目，本项目并未止步于简单的 T检验，而是构建了完整的实验评估框架：

### 1. 数据清洗与质量控制
* **SRM 检验 ：**
    * 使用 **卡方检验** 验证流量分配的均匀性。
    * 结果：P-value > 0.01，确认分流系统无故障，排除了幸存者偏差对实验的干扰。

### 2. 统计推断 
* **假设检验：** 采用 **双尾 Z-test ** 对比两组转化率。
* **置信区间：** 计算差异的 95% 置信区间，确认下界大于 0，量化了最差情况下的正向收益。

### 3. 趋势稳定性分析 
* 绘制 **累计转化率 ** 曲线，通过时间维度监控数据收敛情况，有效排除了 **“新奇效应”** 对早期数据的干扰。
